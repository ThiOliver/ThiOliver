## 📊 Projetos em Destaque

### 🏭 Pipeline de Manutenção Preditiva Industrial
Modelo de Machine Learning para prever falhas mecânicas em equipamentos industriais a partir de dados de sensores. Pipeline completo com tratamento de dados, balanceamento com SMOTE e comparação entre KNN e Árvore de Decisão.

**Stack:** Python, Pandas, Scikit-learn, Imbalanced-learn, Matplotlib, Seaborn
**Resultado:** 92,65% de acurácia

🔗 [Ver projeto](https://github.com/ThiOliver/projeto-analise-de-manutencao-preditiva)

### 📈 SalesInsight PY
Pipeline de análise de vendas B2B para segmentação de clientes e projeção de tendências. Automação de relatórios estruturados (CSV/JSON) e visualização gráfica.

 **Stack:** Python, Pandas, NumPy

🔗 [Ver projeto](https://github.com/ThiOliver/salesinsight-py)
